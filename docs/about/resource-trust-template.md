用于统一站内“工具/外链推荐”的信息质量，减少“只给链接、没有判断依据”的问题。

## 推荐字段（建议每条都写）

- **资源名称**
- **资源类型**（官方文档 / 开源项目 / 教程 / 课程 / 社区帖）
- **适用场景**（适合谁、用来解决什么问题）
- **推荐等级**（高 / 中 / 低）
- **使用门槛**（低 / 中 / 高）
- **成本信息**（免费 / 付费 / 需代理 / 需账号）
- **最后验证日期**
- **替代方案**（至少 1 个）
- **风险提示**（如版本变化快、广告多、可能失效等）

## 评分说明（简化版）

- **高**：来源稳定、内容准确、可复现、维护活跃
- **中**：可用但有局限（版本旧、示例少、门槛较高）
- **低**：仅作参考，需交叉验证

## 可直接复制的模板

```md
### 资源名称

- 资源类型：
- 链接：
- 适用场景：
- 推荐等级：高 / 中 / 低
- 使用门槛：低 / 中 / 高
- 成本信息：免费 / 付费 / 需代理 / 需账号
- 最后验证日期：YYYY-MM-DD
- 替代方案：
- 风险提示：
```

## 示例（简版）

### Docusaurus 官方文档

- 资源类型：官方文档
- 链接：[Docusaurus Docs](https://docusaurus.io/docs)
- 适用场景：需要搭建或维护文档站时查 API 与最佳实践
- 推荐等级：高
- 使用门槛：中
- 成本信息：免费
- 最后验证日期：2026-04-20
- 替代方案：VitePress、MkDocs Material
- 风险提示：版本升级后配置项可能变动
