建议每个模块页面按以下结构组织，保证“新用户能跟得上，老用户能快速定位”。

## 推荐结构（固定顺序）

1. **概览**
   - 这个模块解决什么问题
   - 适合什么阶段的读者
2. **推荐工具/资源**
   - 主推荐（含理由）
   - 备选方案
3. **上手任务**
   - 30-60 分钟可完成的最小实践
4. **常见坑**
   - 高频错误 + 排查思路
5. **进阶路径**
   - 下一步建议阅读/实践
