# Visual Studio Code 安装指南
  
<img src={require('../../../static/img/examples/vscode-logo.webp').default} width="80"/>  


## 下载

- 推荐从 [Visual Studio Code 官网](https://code.visualstudio.com/) 下载 Windows / macOS / Linux 安装包。  
- Windows 也可在 **Microsoft Store** 搜索「Visual Studio Code」安装（发布者为 Microsoft）。  
- **不要在电脑品牌自带的应用商店**（如部分机型的「联想应用商店」等）里安装来源不明的「魔改版」或旧版本，以免捆绑插件与安全隐患。    

  如图：  
![vscode](/img/examples/vscode.webp)

## 配置

1. 安装 **中文（简体）语言包**（扩展名一般为 `MS-CEINTL.vscode-language-pack-zh-hans`），在命令面板中将显示语言切换为中文。  
2. 常用扩展（按需安装即可）：
   - **Python**（Microsoft）
   - **Prettier**（代码格式化）
   - **EditorConfig**（与团队缩进风格一致时很有用）
   - **Live Preview**（静态网页预览）
   - **Stylelint**（写 CSS 时）
   - **AI 代码补全类扩展**：各厂商名称与政策变化较快（如通义灵码、GitHub Copilot 等），请从扩展市场选择**官方发布者**的安装包，并阅读其隐私说明。

## 与本站文档协作

本站正文多为 Markdown，在 VS Code 中安装 **Markdown 相关预览扩展** 即可边写边看渲染效果；若参与本站仓库开发，还需按仓库根目录说明安装 **Node.js** 并执行 `npm install` / `npm run start` 预览 Docusaurus。
