## 项目简介

这是一个面向 **大数据管理与应用专业** 学习者的开源文档站，聚焦“从入门到能用”的学习路径与工具实践。  
内容覆盖基础技能、生成式 AI、网络工具、方法论以及站点建设经验，目标是：

* 新生可快速上手的成长地图
* 可持续更新的学习工具箱
* 可共建、可复用的知识资源池

项目使用 **Docusaurus** 构建，文档以 Markdown/MDX 为主，强调结构清晰、内容实用、可持续维护。

**在线访问地址**：

👉 GitHub Pages: [https://zzming-tjufe.github.io/tjufe_dmresweb_v2/](https://zzming-tjufe.github.io/tjufe_dmresweb_v2/)

## 内容导航

站内内容按模块组织，便于快速定位：

* **开源工具实践**：命令行、Git、Docker、VS Code、环境配置等基础能力。
* **生成式 AI 导论**：ComfyUI、Dify 等工具的入门与应用路径。
* **网络与访问实践**：学习相关网络工具配置与常见问题。
* **新手必读**：Markdown、环境搭建、搜索策略、文档阅读方法。
* **关于与贡献**：项目背景、协作流程、资源可信度模板。
* **更新与发布文档**：更新日志、发布清单、截图基线、回归矩阵。

## 快速开始

### 方式一：在线阅读（推荐）

直接访问上方在线地址，获得完整阅读体验。

### 方式二：本地开发与预览

如果你想本地修改、预览或参与贡献，可按以下步骤运行：

```bash
# 1) 克隆仓库
git clone https://github.com/zzming-tjufe/tjufe_dmresweb_v2.git

cd tjufe_dmresweb_v2

# 2) 安装依赖
npm install

# 3) 启动本地开发服务器（Docusaurus）
npm run start

# 4) 浏览器访问
# http://localhost:3000/tjufe_dmresweb_v2/
```
